## Install PM2

```bash
pnpm add -g pm2
```

## Start Production

```bash
# Build dulu (jika belum)
pnpm run build

# Start semua services dengan PM2
pnpm run start:prod
# atau
pm2 start ecosystem.config.cjs
```

## Commands

```bash
# List process
pm2 list

# Monitor realtime
pm2 monit

# Logs
pm2 logs

# Restart
pm2 restart all

# Stop
pm2 stop all

# Delete
pm2 delete all
```

## Auto-Start on Reboot

```bash
# Setup startup script
pm2 startup

# Save current process list
pm2 save
```

## Access

- Frontend: http://localhost:3001
- Backend: http://localhost:3002
- With Nginx: http://localhost (port 80)
